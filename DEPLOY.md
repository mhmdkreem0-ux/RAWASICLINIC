# 🦷 رواسي لطب الأسنان — دليل النشر على Vercel + Neon

---

## ⚡ الخطوات بالترتيب

---

### الخطوة ١ — إنشاء قاعدة بيانات Neon (مجانية)

1. اذهب إلى **[neon.tech](https://neon.tech)** وسجّل حساباً
2. اضغط **"New Project"** → اختر اسم المشروع: `rawasi-clinic`
3. بعد الإنشاء، اضغط **"Connection Details"**
4. انسخ رابط الاتصال — يبدو هكذا:
   ```
   postgresql://username:password@ep-xxx.us-east-2.aws.neon.tech/neondb?sslmode=require
   ```
5. احفظه — ستحتاجه في الخطوة التالية

---

### الخطوة ٢ — رفع المشروع على Vercel

**الطريقة الأسهل (بدون Git):**

1. اذهب إلى **[vercel.com](https://vercel.com)** وسجّل دخول
2. اضغط **"Add New → Project"**
3. اختر **"Deploy from template"** أو **"Upload"**
4. ارفع المجلد كاملاً (يحتوي على: `server.js`, `index.html`, `package.json`, `vercel.json`)

**أو عبر GitHub:**
```bash
# 1. أنشئ repository على GitHub وارفع الملفات
# 2. في Vercel: Import Git Repository → اختر الـ repo
```

---

### الخطوة ٣ — إضافة المتغيرات البيئية في Vercel

في لوحة Vercel → مشروعك → **Settings → Environment Variables**

أضف هذه المتغيرات:

| الاسم | القيمة |
|-------|--------|
| `DATABASE_URL` | الرابط من Neon (postgresql://...) |
| `JWT_SECRET` | أي نص عشوائي طويل مثل: `rawasi-2025-super-secret-xyz` |

> **ملاحظة**: لا تحتاج `SMTP` أو `GOOGLE` الآن — واتساب يعمل بدونها

---

### الخطوة ٤ — نشر المشروع

في Vercel اضغط **"Deploy"**

بعد دقيقة ستحصل على رابط مثل:
```
https://rawasi-clinic.vercel.app
```

---

### الخطوة ٥ — تهيئة قاعدة البيانات تلقائياً

عند أول زيارة للموقع، يقوم السيرفر تلقائياً بـ:
- ✅ إنشاء جميع الجداول
- ✅ إضافة الأطباء الأربعة
- ✅ إنشاء حساب المدير

**بيانات الدخول الأولى:**
```
البريد:    admin@rawasi.iq
كلمة المرور: admin123
```

> **مهم**: غيّر كلمة المرور بعد أول دخول!

---

### الخطوة ٦ — إضافة المستخدمين

بعد الدخول كمدير:
1. اذهب إلى **المستخدمون** في القائمة الجانبية
2. اضغط **"➕ إضافة مستخدم"**
3. أضف موظفي الاستقبال والأطباء

---

## 🔧 حل المشاكل الشائعة

### ❌ خطأ "الاتصال بقاعدة البيانات فشل"
- تأكد أن `DATABASE_URL` صحيح في متغيرات Vercel
- تأكد أن رابط Neon يحتوي على `?sslmode=require` في النهاية
- في Neon: تأكد أن المشروع نشط (لم يُوقف للخمول)

### ❌ خطأ "غير مصرح" عند تسجيل الدخول
- تأكد من وجود `JWT_SECRET` في متغيرات Vercel
- امسح الـ cookies وأعد المحاولة

### ❌ الموقع يفتح لكن API لا يعمل
- تحقق من Vercel → Functions log لمعرفة الخطأ الحقيقي
- تأكد أن `vercel.json` موجود في المجلد الرئيسي

### ❌ واتساب لا يفتح
- المتصفح قد يمنع النوافذ المنبثقة — اسمح بها أو
- استخدم زر 📱 في شاشة تفاصيل الموعد
